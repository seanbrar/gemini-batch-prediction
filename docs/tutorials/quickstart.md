# Quickstart

This short tutorial gets you from zero to a basic batch run.

## 1) Install

```bash
make install-dev
```

## 2) Run an example

```bash
python examples/simple_demo.py
```

## 3) Explore batch processing

```bash
python examples/batch_text_demo.py
```
